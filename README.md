# 🎬 Friends Favorite Movies — Part 2
A SwiftUI multi-screen app series — Part 2 adds the ability to create, edit, and delete movie entries.

---

## 🤔 What this is
This is Part 2 of the Friends Favorite Movies series. Building on Part 1, this episode adds full CRUD functionality — users can add new movies, edit existing entries, and delete them from the list. It introduces SwiftUI forms, sheet presentation, and state management patterns for editable data.

## ✅ Why you'd use it
- **SwiftUI forms & sheets** — Shows how to present an add/edit form in a sheet and bind it to the app's data model
- **CRUD with @State** — Demonstrates create, read, update, and delete operations on an in-memory array with SwiftUI state
- **Edit mode** — Covers `EditButton`, swipe-to-delete, and reordering in a SwiftUI List

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/FriendsFavoriteMovies-Part-2.git
cd FriendsFavoriteMovies-Part-2
```

### 2. Open in Xcode
Double-click `FriendsFavoriteMovies1.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.
